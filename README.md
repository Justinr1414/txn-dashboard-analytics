# Transaction Analytics Dashboard (Power BI)

## Overview

This project highlights a transaction analytics dashboard designed to track performance trends across multiple time periods, including daily, weekly, monthly, and quarterly views.

The dashboard focuses on identifying changes in transaction volume, measuring growth, and providing clear insights into performance fluctuations over time.

## Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Excel (data preparation)

## Key Features

* Dynamic time-based analysis (daily, weekly, monthly, quarterly)
* Period-over-period comparison (e.g., week-over-week, month-over-month)
* KPI tracking for transaction growth and decline
* Interactive filtering for deeper analysis

## Technical Highlights

* Built custom DAX measures for rolling period comparisons
* Designed dynamic calculations for transaction variance and trend analysis
* Implemented interactive visuals and bookmarks for user-friendly navigation

## Note

This project is based on professional experience and has been adapted to remove any proprietary or sensitive data.
